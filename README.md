## Setup

<p>1. Clone the repository.</p>

```bash
git clone https://github.com/demon-bixia/Avian.git
```

<p>2. Install dependencies.</p>

```bash
npm install
```

<p>3. Run the development server.</p>

```bash
npm run dev
```
